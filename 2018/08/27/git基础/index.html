git基础

Posted on 2018-08-27

git init

新建一个仓库

创建一个文件夹，打开，输入命令

git init

这样就初始化了一个本地仓库。

git add

添加文件到暂存区

创建一个文件，输出命令

git add <文件名>

这样就将该文件添加到暂存区

git commit -v

提交时显示所有diff信息
