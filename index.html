JSONP

请求方：frank.com 的前端程序员（浏览器）
响应方：jack.com 的后端程序员（服务器）

  1. 请求方创建 script，src 指向响应方，同时传一个查询参数 ?callbackName=yyy
  2. 响应方根据查询参数callbackName，构造形如yyy.call(undefined, ‘你要的数据’)这样的响应
  3. 浏览器接收到响应，就会执行 yyy.call(undefined, ‘你要的数据’)
  4. 那么请求方就知道了他要的数据

这就是JSONP，动态标签跨域请求技术。

如何实现页面内跳转

发表于 2018-09-14

1. 如何实现点击链接跳转页面内指定位置？

我们都知道，通过设置a标签的锚点可以实现页面内跳转。像这样<a href="#xxx"></a>这时点击a标签便跳转到id为xxx的元素。
但是这样存在一个问题，如果有导航栏的话，导航栏会遮盖住上面一部分，很不美观，为了避免这种情况，那么我们就需要点击a标签时，使其跳转到我们指定的高度，如何操作呢？

当然我们还是使用锚点来跳转，但是首先我们要阻止a标签的默认行为也就是点击跳转。因为跳转的位置不是我们想要的。
使用preventDefault()方法可以阻止默认行为。

然后我们获取用户点击的a标签

接着通过该a标签获取跳转内容的高度

<body>
  <div id="xxx" class="xxx" style="width: 100px;height: 100px;background: green;"></div>
</body>
  <script>
    xxx.style.position = 'relative'
    xxx.style.left = 0

    var n = 0
    var id = setInterval(()=> {  //设置定时任务
      n=n+1
      xxx.style.left = n + 'px'
      if(n>200){
        window.clearInterval(id)  //取消定时任务
      }
    },20) //每20ms动一次
  </script>

    let aTags = document.querySelectorAll('nav.menu > ul > li > a')
    for (let i = 0; i < aTags.length; i++) {
    aTags[i].onclick = function (x) {
        x.preventDefault()
        let a = x.currentTarget
        let href = a.getAttribute('href')
        let element = document.querySelector(href)
        let top = element.offsetTop

        let n = 20  //一共动多少次
        let duration = 500 / n  //多长时间动一次
        let currentTop = window.scrollY  //获取当前高度
        let targetTop = top - 80 // 获取目标高度
        let distance = (targetTop - currentTop)/ n  //当前高度减去目标高度，也就是要动的距离
        let i = 0
        let id = setInterval(()=> {
            if(i===n){
                window.clearInterval(id)  // 到达目标时停止
                return
            }
            i=i+1
            window.scrollTo(0,currentTop + distance * i)
        },duration)
    }
}

如何声明一个函数

JavaScript有五种方法声明一个函数

1. function命令

function声明的代码区块，就是一个函数。function后面是函数名，函数名后面是一对圆括号，里面是传入函数的参数。函数体放在大括号里面。

2,3. 函数表达式

除了function命令声明函数，还可以采用变量赋值的方法

变量赋值声明函数，有两种方法，一种是匿名函数赋值，一种是具名函数赋值。
（1）匿名函数赋值

这种写法将一个匿名函数赋值给一个变量。这是，这个匿名函数又称函数表达式，因为赋值语句的右侧只能放表达式。

(2)具名函数赋值

采用函数表达式声明函数时，function命令后面不带有函数名。如果加上函数名，该函数名只在函数体内部有效，在函数体外部无效。

上面代码在函数表达式中加入了函数名b，这个b只能在函数体内部使用，其他地方都不可用。
这种写法的用处有两个:

  • 可以在函数体内部调用自身
  • 方便除错。除错工具显示函数调用栈时，将显示函数名，而不再显示这里是一个匿名函数。

4. Function 构造函数

  • 上面代码中，Function构造函数接受三个参数，除了最后一个参数是a函数的函数体，其他参数都是a函数的参数。
  • 你可以传递任意数量的参数给Function构造函数，只有最后一个参数会被当成函数体，如果只有一个参数，该参数就是函数体。

5. 箭头函数

箭头函数其实就是函数表达式方法中，匿名函数赋值的省略版本。

如果箭头函数的代码块部分只有一句，则可以省略return和大括号。注意，必须同时省略。

DOM解析

发表于 2018-09-04

DOM是啥？

MDN是这样解释的:

文档对象模型 (DOM) 是HTML和XML文档的编程接口。它提供了对文档的结构化的表述，并定义了一种方式可以使从程序中对该结构进行访问，从而改变文档的结构，样式和内容。DOM将文档解析为一个由节点(node)和对象（包含属性和方法的对象）组成的结构集合。简言之，它会将web页面和脚本或程序语言连接起来。

OK,翻译成人话就是，本来我们是不能用JS操作html页面的，因为页面只是文本，浏览器将html页面变成DOM，内容没有变，形式变成了对象,我们就可以用JS操作他了。web页面和DOM是一个东西，但是表现形式不同。就像水和冰，一个是文档一个是文档对象模型(DOM)。
